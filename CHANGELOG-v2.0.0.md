# IP Discovery V2.0.0 — 变更日志

> **发布日期**：2026-04-05
> **版本**：2.0.0（Major — 全面重构升级）

---

## 一句话总结

**从「聊天式六维度问卷」到「专业咨询级深度访谈系统」—— 引入咨询工具箱穿透力、全程可视化仪表盘、五级递进交付物、以及编辑式杂志风格的 HTML 可视化报告。**

---

## 本次迭代全景

### 🧠 核心方法论升级

| 维度 | V1 | V2 |
|:-----|:---|:---|
| 访谈深度 | 聊天式引导，单层问题 | **三层问题库**（信号捕捉→深度挖掘→顿悟触发）+ 5大咨询技术工具箱 |
| 过程感知 | 开头提一次六维表后消失 | **每轮输出实时仪表盘**（探索状态/成熟度评分/关键发现/下一步方向） |
| 交叉分析 | 简单一致性说明 | **完整分析框架**：一致性矩阵 + 张力全景图 + 协同效应 + 定位强度评估 |
| 交付物 | 最终一次性 Markdown | **五级递进交付物**：画像卡→能力图谱→定位草图→方案集→双模报告 |

### 📦 文件变更清单

| # | 文件 | 操作 | 核心变化 |
|---|:-----|:----:|:---------|
| 1 | `SKILL.md` | **重写** | 咨询工具箱 + 仪表盘机制 + 五级交付物 + 交叉框架（~550行） |
| 2 | `references/question-bank.md` | **重写** | 每维度新增第三层深度挖掘 + 咨询技术速查（~320行） |
| 3 | `assets/ip-profile-report.html` | **新建** | HTML 可视化报告模板，编辑式杂志风格（~900行） |
| 4 | `assets/ip-profile-template.md` | **重构** | 10章→11章完整报告（含竞品对标/内容策略/受众画像） |
| 5 | `references/competitive-analysis.md` | **新建** | 竞品对标方法论（选择→拆解→坐标→机会识别） |
| 6 | `commands/ip-discovery.md` | **重写** | V2 风格开场白（含方法论介绍） |
| 7 | `plugin.json` | **更新** | 版本号 1.0.0 → 2.0.0，描述更新 |

### 🎨 HTML 报告视觉重设计（去 AI 味）

| 维度 | 改前 | 改后 |
|:-----|:-----|:-----|
| 风格 | SaaS Dashboard 暗色风 | 编辑式杂志报告 — 温暖纸张质感 |
| 字体 | 系统字体栈 (Segoe UI / PingFang SC) | Noto Serif SC + DM Sans + JetBrains Mono |
| 配色 | 深色底 + indigo/purple 紫色渐变 | 暖灰纸底 + 琥珀主调 + 墨绿辅助（零紫色） |
| 排版 | emoji 图标 + 圆角卡片堆叠 | 编号章节(01/02…) + 衬线体 + 引用式排版 + 微边框 |
| 动效 | pulse 背景动效 + hover 上浮 | Staggered reveal 入场 + max-height 折叠动画 + 进度条缓动 |

### 🔧 新增能力清单

- [x] **咨询技术工具箱**：5 Whys 深挖 / 反证挑战 / 极端场景测试 / 认知冲突利用 / 顿悟时刻设计
- [x] **实时维度仪表盘**：6 维度 × {状态标签 + 成熟度 0-10 分 + 关键发现 + 下一步}
- [x] **五级递进交付物**：🧭初始画像卡 → 🔬能力深度图谱 → 🎨差异化定位草图 → 🎯IP方案集 → 📊可视化报告
- [x] **HTML 可视化报告**：Canvas 雷达图 + 可折叠维度面板 + 方案三选一卡片 + 竞品地图 + 30天时间线
- [x] **竞品对标体系**：选择原则 → 六维拆解 → 坐标定位 → 差异化机会识别
- [x] **第三层深度问题库**：每个维度 8-10 道深度挖掘问题（含天赋区探测/异端观点/价值产品化等）

### ⚠️ 已知待完善项

- `ip-quick.md` 快速评估命令尚未同步升级为 V2 风格
- `content-strategy` / `audience-analysis` 技能与主流程的融合优化
- HTML 报告模板需经真实数据填充测试验证效果
