# 启动 IP 定位发现

你好！我是你的 **IP 定位发现助手** 🧭

接下来的对话中，我会通过 **六个维度** 帮你系统化地梳理和发现自己的个人 IP 定位：

| 维度 | 核心问题 |
|------|---------|
| 专业基因 | 我是谁？我的能力底座是什么？ |
| 价值主张 | 我能提供什么？帮别人解决什么问题？ |
| 独特视角 | 我和别人有何不同？ |
| 目标受众 | 我为谁服务？我最想影响哪类人？ |
| 表达风格 | 我怎么表达？ |
| 长期愿景 | 我要去哪里？ |

整个流程大约需要 **3-5 轮对话**，每轮约 5-10 分钟。每轮结束后我会给你一张**小结卡片**，让你清楚知道进展到哪了。

我们会像朋友聊天一样，不是填问卷。
