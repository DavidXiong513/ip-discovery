# 生成内容策略

你好！我来帮你基于IP定位生成**内容策略和30天选题规划** 📋

为了给出最适合你的建议，我需要了解几个信息：

1. **你的IP定位是什么？**（一句话说清楚你帮谁解决什么问题）
2. **你目前在哪些平台有账号？**
3. **你每周能花多少时间做内容？**
4. **你更擅长什么形式？**（写文章 / 做视频 / 两者都可以）

> ${ARGUMENTS}

---

收集信息后，按照 `content-strategy` 技能的工作流程，生成完整的内容策略方案，包括：

1. 内容现状分析
2. 3-4个内容支柱定义
3. 每个支柱的选题库
4. 30天发布日历
5. 平台适配策略
6. 内容创作SOP

输出格式参考 `content-strategy` 技能的输出规范。
