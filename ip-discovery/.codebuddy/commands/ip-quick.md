# 快速 IP 定位评估

你好！来做个**快速IP定位扫描** ⚡

不需要长篇大论，回答下面几个问题就行，大概2分钟：

1. **你现在做什么工作/事情？做了多久？**
2. **你最擅长帮别人解决什么问题？（哪怕很小的事）**
3. **你和朋友/同事最不一样的地方是什么？**
4. **你最想让哪类人看到你的内容？**
5. **如果用3个词形容你自己，你会选哪3个？**

> 直接列出编号回答就行，想到什么说什么。

${ARGUMENTS}

---

收集到回答后，按照 `ip-discovery` 技能的快速评估模式，直接输出简化版画像：

```
# [IP名称] — 快速IP定位评估

## IP一句话定位
> "[你是谁] + [用什么方式] + [帮谁] + [解决什么问题]"

## 核心标签
[3-5个标签]

## 🔼 最大优势
[1-2句话]

## 🔽 最大风险/盲点
[1-2句话]

## 下一步行动建议
1. [具体可执行的第一个行动]
2. [第二个行动]
3. [第三个行动]
```

如果用户想要更详细的完整画像，引导其使用 `/ip-discovery` 启动完整流程。
