# 竞品对标分析方法论 V2

> **使用说明**：此文档为 IP Discovery V2 竞品对标分析的参考方法论。
> 在访谈第5轮（可选）或最终报告生成阶段使用，为用户的IP定位提供**外部视角和市场验证**。

---

## 为什么需要竞品对标

在 V1 版本中，整个 IP 发现流程 100% 聚焦用户自身——这是一个**重大盲点**：

| 问题 | 后果 |
|:-----|:-----|
| 没有外部参照系 | 用户不知道自己的定位在市场中处于什么位置 |
| 缺乏差异化验证 | "独特性"是否真的独特？还是自以为独特 |
| 忽视市场空白 | 可能错过竞品们都忽略的机会区域 |
| 无法评估竞争强度 | 不知道选定的方向是蓝海还是红海 |

**核心原则**：我们不做竞品分析的目的是"模仿"，而是为了：
1. **验证差异化的真实性**
2. **发现市场空白机会**
3. **帮用户找到坐标系中的位置**

---

## 第一步：选择对标 IP

### 选择原则

至少选择 **3 个** 对标 IP，覆盖以下三种类型：

#### 类型 A：头部标杆（同赛道 Top）

- **定义**：你所在领域/方向的头部 IP，粉丝量大、影响力强
- **目的**：了解"天花板"长什么样，行业最高标准
- **数量**：1 个
- **示例**：想做效率提升博主 → 选"头部效率博主"

#### 类型 B：中腰部同类（直接参考）

- **定义**：和你目标受众相似、内容方向相近的中腰部 IP
- **目的**：最直接的对比参照——你们之间有什么不同
- **数量**：1-2 个
- **示例**：同是做职场成长但风格不同的博主

#### 类型 C：差异化参照（间接启发）

- **定义**：不在同一赛道但有一些可借鉴之处的 IP
- **目的**：跨领域学习——"TA的做法能否迁移到我的领域"
- **数量**：0-1 个
- **示例**：一个表达方式特别好的美食博主 → 学习其叙事方式

### 如何获取对标信息

在访谈过程中自然地询问：

```
"在你想要做的这个领域里，
 有没有你特别关注甚至崇拜的博主/IP？
 他们叫什么名字？
 你觉得他们做得最好的地方是什么？"
```

或者让用户主动提供：

```
"如果让你列出3个你觉得做得很好、
 或者你想成为的那种人，
 你会列谁？为什么？"
```

> ⚠️ 如果用户说不出任何对标 IP，这可能意味着：
> - 该领域太新/太小众（可能是机会也可能是风险）
> - 用户对市场缺乏了解（需要在报告中标注）
> - 建议用 WebSearch 辅助调研

## 第二步：六维度拆解对标 IP

对每个竞品 IP，按照六维度模型进行拆解评分：

### 拆解模板

| 维度 | 分析问题 | 评分(0-10) | 关键观察 |
|:-----|:---------|:-----------|:---------|
| **专业基因** | TA的专业背景是什么？能力底座有多深？ | _/10 | |
| **价值主张** | TA主要解决什么问题？价值主张明确吗？ | _/10 | |
| **独特视角** | TA和同类最大的不同是什么？有反常识观点吗？ | _/10 | |
| **目标受众** | TA主要服务谁？受众画像清晰吗？ | _/10 | |
| **表达风格** | TA的风格DNA是什么？辨识度高吗？ | _/10 | |
| **长期愿景** | 能看出TA的方向感吗？有边界意识吗？ | _/10 | |

### 评分注意事项

- **评分基于公开可观察的信息**（不是猜测），不确定的标注"?"
- **关注"感知到的"而非"真实的"** —— IP的本质是感知管理
- **记录关键观察**而不只是打分 —— 数字背后的洞察更重要

### 快速拆解提示词

当需要快速分析一个竞品时，可以用以下结构化提问：

```
请分析 [竞品IP名称] 的六维度构成：

1. 专业基因：TA看起来最擅长的领域/技能是什么？
2. 价值主张：TA的内容主要给读者带来什么价值？
3. 独特视角：TA和其他同类账号最大的区别在哪里？
4. 目标受众：从TA的内容来看，TA最想影响谁？
5. 表达风格：用一个词形容TA的风格。标志性特征是什么？
6. 长期愿景：从TA的内容走向看，TA想往哪个方向发展？

最后给出综合评价：TA最强的一维和最弱的一维分别是什么？
```

## 第三步：构建市场定位坐标

### 二维定位坐标系

根据用户所在的领域特点，选择合适的坐标轴：

#### 常用坐标轴组合

| 坐标轴 X | 坐标轴 Y | 适用场景 |
|:---------|:---------|:---------|
| 广度 ↔ 深度 | 专业 ↔ 大众 | 知识类/教育类 IP |
| 理性 ↔ 感性 | 实用 ↔ 启发 | 成长/心理类 IP |
| 小众 ↔ 大众 | 高价 ↔ 免费 | 商业/咨询类 IP |
| 信息 ↔ 观点 | 娱乐 ↔ 教育 | 泛内容类 IP |

### 坐标图绘制方法

```
           [Y轴标签]
            ↑
      [竞品A] ●
                \
     你 ●—————● [竞品B] ————→ [X轴标签]
              /
    [竞品C] ●
```

**关键输出**：
1. 你在坐标中的位置描述
2. 与每个竞品的距离关系（近=直接竞争，远=错位竞争）
3. 图中的"空白区域"（没人占据的位置）

## 第四步：差异化机会识别

### 三层机会分析法

#### 第一层：空白机会（Blue Ocean）

> **"竞品们都忽略了什么？"**

检查清单：
- [ ] 有没有任何细分受众被所有人忽视了？
- [ ] 有没有某种内容形式在这个领域还没人做好？
- [ ] 有没有某个观点角度是所有人都回避的？
- [ ] 价格/付费模式上有没有空白地带？

#### 第二层：可借鉴点（Learn & Adapt）

> **"TA们做得好的地方，我怎么做会不一样？"**

| 可借鉴的方向 | 竞品来源 | 你的差异化做法 |
|:------------|:--------:|:--------------|
| 内容系列设计 | 竞品A | [如何做出不同] |
| 互动方式 | 竞品B | [如何做出不同] |
| 个人故事运用 | 竞品C | [如何做出不同] |
| 视觉/包装 | 竞品A | [如何做出不同] |

⚠️ **铁律**：可以借鉴思路，但绝不照搬执行

#### 第三层：不可照搬的差异（Your Moat）

> **"有些东西竞品有但你学不来——那就是你的护城河"**

这些通常来自：
- 独特的人生经历组合
- 特定的专业背景交叉
- 性格特质带来的天然风格
- 特殊的价值观或立场

## 第五步：输出格式规范

### 在最终报告中的呈现位置

竞品对标分析应在最终报告中作为**独立章节**呈现（Markdown模板第六章 / HTML报告对应区块）。

### 最小输出要求

即使时间有限无法深入分析，也必须包含：

1. ✅ **3个竞品名称 + 类型标签**（头部标杆/中腰部/差异化参照）
2. ✅ **简版六维度对比表**（4行×7列表格）
3. ✅ **你的坐标位置一句话描述**
4. ✅ **至少1个差异化机会**

### 完整输出（推荐）

包含以下所有部分：
1. ✅ 对标 IP 选择及理由
2. ✅ 六维度详细拆解（3个IP）
3. ✅ 市场定位坐标图（文本可视化）
4. ✅ 差异化机会清单（≥3条）
5. ✅ 可借鉴点 vs 不可照搬差异对照表
6. ✅ 竞品分析的局限性和风险提示

## 常见情况处理

### 用户说"没有对标"

```
回应策略：
"这本身就是一个有意思的信号——可能说明这个方向很新、
 或者你还没有开始系统性地研究市场。
 我们可以先跳过这一步，
 但我建议你在正式开始之前花1-2天看看这个领域的玩家都有谁。
 我也可以帮你做一些基础的调研。"
```

### 用户只想模仿某个人

```
回应策略：
"好，让我们来做一个深度拆解。
 不是要复制TA，而是理解TA成功的底层逻辑——
 哪些是可以学的，哪些是学不来的，
 还有哪些是你能做得不一样的。
 这才是有价值的对标分析。"

然后启动完整的竞品六维度拆解 + 差异化空间寻找
```

### 找不到合适竞品（领域太小众）

```
处理方案：
1. 放宽范围到相邻领域找间接竞品
2. 用更通用的维度对标（如：表达风格、受众互动方式）
3. 在报告中明确标注"该领域尚无直接竞品 = 先发优势/市场未验证"
4. 建议用户关注相关但不完全相同的领域
```

## 与主流程的集成时机

| 访谈阶段 | 集成方式 |
|:---------|:---------|
| 第1-2轮 | 收集对标线索（"有没有你关注的博主?"） |
| 第3轮 | 开始形成差异化认知时引入初步对标思考 |
| 第4轮 | 方案生成时用竞品数据验证方案的可行性 |
| 第5轮（可选） | 正式进行完整竞品对标分析 |
| 最终报告 | 作为独立章节完整呈现 |
